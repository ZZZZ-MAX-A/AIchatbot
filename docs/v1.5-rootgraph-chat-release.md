# v1.5 RootGraph Chat 封版记录

封版日期：2026-07-06

封版 tag：`v1.5`

封版结论：`v1.5` 已达到 feature complete，可以作为 RootGraph 普通聊天接入与可观测性阶段的稳定基线。后续不再向 `v1.5` 追加新功能，除非是明确的回归修复或文档勘误。

## 本版目标

`v1.5` 的目标不是扩大机器人的功能面，而是把 QQ 运行时的关键入口收束到更清晰的图结构里：

```text
普通聊天 -> RootGraph CHAT -> 既有聊天链路
/agent   -> RootGraph MAIN_AGENT -> MainAgentGraph
```

RootGraph 在本版承担统一的调度、权限前置、路由、上下文层级标记、提交状态汇总和观测职责。具体聊天生成、记忆注入、视觉解析、语音发送和 MainAgent 工具执行仍由各自子图或既有链路负责。

## 已完成能力

本版完成了以下主线：

```text
RootGraph Runtime 节点序列补齐：
  normalize_event
  load_actor_context
  hard_policy_gate
  route_intent
  build_runtime_context
  dispatch_capability
  commit_side_effects
  render_response

普通聊天接入：
  非语义语音的普通聊天经 RootGraph CHAT 分发。
  CHAT handler 复用既有 legacy/chat graph 聊天链路。
  RootGraph 外层不重复发送已经由聊天链路发送过的 QQ 回复。

权限前置：
  普通聊天 preflight 写入 chat_access_policy artifact。
  RootGraph hard_policy_gate 读取 chat_access_policy。
  allow_dispatch=false 时 CHAT 不进入聊天 handler。

提交观测：
  聊天链路写入 chat_commit artifact。
  RootGraph commit artifact 汇总 reply、voice、persist、trial、compression、tts、image_deferred 等状态。

错误观测：
  RootGraph 新增结构化 error artifact。
  handler 异常、policy 拒绝和 state.error 都有非正文观测摘要。

/agent 观测：
  /agent RootGraph 最近观测 可查看最近 CHAT 快照。
  /agent MainAgent 最近观测 可查看 MainAgent 执行摘要。
  /agent 聚合诊断入口已合并视觉、RAG、错误、RootGraph、MainAgent 状态。

视觉可靠性：
  VISION_NUM_CTX 默认提升到 16384。
  Ollama vision 请求写入 options.num_ctx。
  @@@@@@ 等低质量重复符号输出会被判定为视觉失败，不写入图片上下文。
  /视觉状态 新增真实 qwen2.5-vl 推理自检。

MemoryRAG 可靠性：
  /记忆状态 和 /RAG状态 新增真实 bge-m3 embedding 自检。
  自检只使用固定测试文本，不读取用户内容，不写数据库，不重建索引。

MainAgent Route B 边界：
  owner_read_command 保持只读。
  owner_write_command 继续要求审批。
  agent_task_command 仍为本地确定性控制面，不暴露给 LLM。
  shell、任意文件写入、任意数据库写入继续不开放。
```

## QQ 可用诊断入口

封版后建议优先使用这些命令定位运行问题：

```text
/agent 诊断一下 Ollama
/agent 看一下视觉和记忆状态
/agent 最近图片和 RAG 有没有问题
/agent RootGraph 最近观测
/agent MainAgent 最近观测
/视觉状态
/记忆状态
/RAG状态
/最近错误
```

其中 `/agent 诊断一下 Ollama` 会返回聚合诊断，范围包括：

```text
视觉链路：
  Ollama 地址
  Ollama 服务
  视觉模型
  视觉上下文
  qwen 真实推理自检

RAG/Embedding：
  RAG 开关
  聊天注入
  向量服务
  向量模型
  bge-m3 真实 embedding 自检
  索引和向量记录统计

最近错误：
  最近 error log 摘要

RootGraph：
  最近 CHAT 路由、policy、context、commit、shadow 摘要

MainAgent：
  最近 MainAgent route/action/tool/policy 摘要
```

## 运行边界

本版明确保持以下边界：

```text
普通聊天不会触发 MainAgent ToolRegistry。
普通聊天不会读取 ProjectDocRAG。
ProjectDocRAG 只通过 /agent dev_context 或开发侧命令进入。
MemoryRAG 可以进入普通聊天上下文，但 embedding 失败时应降级，不应阻断普通聊天。
/agent 只允许 owner private 入口。
非 owner 不进入 MainAgentGraph。
写操作必须走 approval。
高风险旧命令不被迁移进 /agent 语义工具。
RootGraph 只做调度和观测，不替代子图内部职责。
```

继续不开放：

```text
shell 工具
任意文件写入
任意数据库写入
任意 QQ 外发
多步自主 agent loop
通用真实任务执行链路
自动长期记忆写入
角色卡自动修改
清空全部上下文语义化
删除长期记忆语义化
```

## 已知风险

```text
视觉模型依赖本地 Ollama 11434 和 qwen2.5vl:3b。
Ollama 托盘版、模型目录或 mmproj 路径异常时，视觉仍可能失败。
bge-m3 依赖本地 Ollama embedding 服务；失效时 RAG 检索会受影响。
/agent 聚合诊断会真实调用视觉推理和 embedding 自检，可能耗时 10-15 秒。
语义语音入口尚未完全纳入 RootGraph CHAT。
部分旧固定命令仍由 NoneBot matcher 兜底处理。
RootGraph 目前汇总 commit 状态，但尚未接管所有副作用实际执行位置。
```
